# How to setup locally and run project

### 1. Clone repository

```
git clone https://github.com/pg07codes/fokus
```

### 2. Enter into cloned repo

```
cd fokus
```

### 3. To install dependencies 

```
npm install
```

### 4. To run development server

```
yarn dev 
```


### 5. To create distribution bundle

```
yarn build
```
